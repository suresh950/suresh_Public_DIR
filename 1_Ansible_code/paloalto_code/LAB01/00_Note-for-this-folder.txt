gather_subset

-
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Default:

["!config"]

Scopes what information is gathered from the device. 
Possible values for this argument include all, system, session, interfaces, ha, routing, vr, vsys and config. 
You can specify a list of values to include a larger subset. 
Values can also be used with an initial ! to specify that a specific subset should not be collected. 
Certain subsets might be supported by Panorama.
